# lycan

Scaffold du projet "lycan" — link-in-bio, mail & chat.

Ce dépôt contient un scaffold React + Vite avec React Router v6, configuration pour déploiement sur Cloudflare Pages/Workers, et un exemple de Worker pour générer des Open Graph dynamiques pour les profils.

Commandes:

- Installation: npm install
- Dev: npm run dev
- Build: npm run build
- Preview build: npm run preview

Structure principale:

- index.html
- src/
- cloudflare-worker/og-worker.js (exemple)
